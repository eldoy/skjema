# Skjema

Generate forms and actions.

Made for the [Waveorb Web Application Development Framework](https://waveorb.com)

### Install

```
npm i skjema
```

### Usage

```js
const skjema = require('skjema')

// Generate form
skjema.form()

// Generate form field
skjema.field()

//




```
