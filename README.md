# Skjema

Low code CMS System.

Generate your entire content admin from YAML files.

### Install

```
npm i -g skjema
```

Then use like this from your application directory:
```
# Start development server
skjema dev

# Build your application
skjema build

# Deploy application
skjema deploy
```

### Development

Start the development server like this:
```
skjema dev
```
